# 888: Merge Rush

A satisfying number-merge puzzler where you combine tiles to reach 8 → 88 → 888!

## How to Play

1. **Open the game**: Simply open `index.html` in any modern web browser
2. **Drag tiles**: Swipe or drag tiles onto adjacent tiles of the same value to merge them
3. **Create combos**: Chain merges quickly to build up an ×8 multiplier
4. **Reach 888**: Merge two 88 tiles to create the ultimate 888 tile and trigger an infinity blast!

## Game Features

### Core Gameplay
- **8×8 board** with tiles dropping from the top
- **Merge mechanics**: 1+1→2, 2+2→3... 7+7→8, 8+8→88, 88+88→888
- **888 Blast**: Creates a massive explosion clearing 3×3 area and all 1s/2s
- **Combo system**: 8-second timer with ×8 maximum multiplier

### Obstacles
- **Stone 8s**: Immovable blockers - destroy by merging next to them 3 times
- **Lock tiles**: Unlock after any 8-tile merge on the board

### Boosters
- **Flip 8**: Rotate any 3×3 area (coming soon)
- **Split**: Downgrade one tile by -1 (coming soon)
- **Shockwave**: Clears the lowest row

### Game Modes
- **Classic (Endless)**: Play until the board fills
- **Time Attack (88s)**: Highest score in 88 seconds
- **Daily 8**: One fixed seed level per day

### Meta Features
- **Daily Quests**: Complete objectives for booster rewards
- **Streak System**: Visit daily to maintain your streak
- **Leaderboards**: Compete for the highest scores
- **Settings**: Sound, vibration, high contrast, and reduced motion options

## Controls

### Mobile/Touch
- **Drag & Drop**: Touch and drag tiles to merge
- **Tap Boosters**: Tap booster buttons to activate

### Desktop
- **Mouse**: Click and drag tiles to merge
- **Keyboard**: Arrow keys for navigation (enhanced keyboard support coming soon)
