## 🎮 Game Overview

Tiger Treasures is a match-3 puzzle game where players swap adjacent gems to create matches of three or more identical gems. With its tiger theme and mobile-first design, it offers an engaging puzzle experience with smooth animations and intuitive touch controls.

## ✨ Features

### 🎯 Core Gameplay
- **8x8 Grid**: Classic match-3 gameplay on a spacious grid
- **Touch Controls**: Optimized for mobile with responsive touch interactions
- **Cascade Matching**: Automatic combo detection for chain reactions
- **Move-Based**: Strategic gameplay with limited moves per level

### 🎨 Visual Design
- **Tiger Theme**: Beautiful orange and gold color scheme
- **7 Gem Types**: Diverse gems including Tiger (🐅), Ruby (💎), Emerald (💚), Sapphire (💙), Diamond (💍), Topaz (🟡), and Amethyst (💜)
- **Smooth Animations**: Falling gems, match effects, and selection highlights
- **Responsive Design**: Works perfectly on all mobile screen sizes

### 📱 Mobile Optimization
- **Touch-Friendly**: Large tap targets and intuitive gestures
- **No Zoom Issues**: Prevents accidental zooming and context menus
- **Portrait/Landscape**: Adapts to both orientations
- **Performance**: Optimized for smooth gameplay on mobile devices

### 🎲 Game Mechanics
- **Level Progression**: Increasing difficulty with higher target scores
- **Scoring System**: Points based on match size (3-match: 100pts, 4-match: 200pts, etc.)
- **Power-ups**: Shuffle and Hint systems to help players
- **Progress Tracking**: Visual progress bar and detailed statistics

## 🚀 How to Play

1. **Start**: Open `index.html` in your web browser
2. **Select**: Tap a gem to select it (highlighted with orange border)
3. **Swap**: Tap an adjacent gem to swap positions
4. **Match**: Create lines of 3+ identical gems horizontally or vertically
5. **Score**: Reach the target score before running out of moves
6. **Progress**: Complete levels to unlock higher challenges

### 🛠️ Game Controls

- **🔄 Shuffle**: Reorganize the board (costs 1 move)
- **💡 Hint**: Highlights a possible match
- **⏸️ Pause**: Pause the game anytime
- **🎯 Progress Bar**: Shows progress toward target score
